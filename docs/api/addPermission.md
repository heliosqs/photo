# Add Permission

Allows a user to get permissions to access an album

**URL** : `/albumauth/`

**Method** : `POST`

**Authentication** : NO

## Successful Response

**Code** : `200 OK`

**Example**

```json{
  "object": {
    "Added Successfully",
    "error": null,
    "successful": true
  },
  "error":null,
  "successful":true
}
```
